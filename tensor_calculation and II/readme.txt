Здесь будут наработки по программе с ИИ
Суть программы - обработка фотографий(картинок) и ответ, что на ней изображено
Библиотека для ИИ - TensorFlow
Возможно потребуется парсинг сайтов для облегчения поиска картинок и быстроты обучения сети

Вспомогательные компоненты:
Последние драйвера Geforce с поддержкой CUDA 10+
CUDA 11.2
CUPTI
cuDNN SDK 8.1
TensorRT 6.0
